# React Portfolio Website

This is my personal portfolio website built using **React.js** to showcase my projects, skills, and education.

- Modern and responsive UI
- Component-based architecture using React
- Smooth animations with Framer Motion
- Clean and structured layout

## Tech Stack
- React.js
- JavaScript
- HTML & CSS
- Framer Motion
